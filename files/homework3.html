Forecasting the Second Hand House Sales in Turkey¶

Mehmet Yusuf Kaya

Introduction¶

The goal of this homework is to predict the number of second hand house sales in Turkey in a mounthly basis. To predict accordingly, the data of the number of second hand house sales in Turkey and the interest rates applied for house credits from 2013.01 to 2020.11 is taken from EVDS web site. Because most of the people take credits from banks to purchase a house, I used interest rate as an input for my analysis. With the regression model, I predict the number of second hand house sale in Turkey for last month of 2020 and twelve months for 2021. These predictions will be checked when the actual data would be available at TCMB EVDS web site.

Conclusion¶

In this homework my aim is to predict the second hand home sales in Turkey for December 2020. To do that I used the past data between 2013 to 2020 which are number of second hand home sales and interest rate for mortgate credits on a monthly basis. Data is provided by EVDS TCMB.¶

The first part of this work is about data manipulation. Dates are set as a datetime. Missing values are filled and the time interval that I want to used is selected. Also I created different dataframes for the possibility to use in the model building section.¶

In the second part I checked the data by looking the graphs and ACF's. Then I build different models. By analysing the models statistically I select the model that has the highest R-Squared(0.897).¶

In the last part I created the December-2020's dataframe and put the corresponding data into it. After that by using the best model I predict the December 2020 number of second hand home sales in Turkey. Also I plotted the graph of actual vs predicted data by using model.¶
